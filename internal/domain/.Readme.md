# 🧠 `internal/domain/`

## Назначение

`domain` содержит **чистые бизнес-сущности** и **доменные интерфейсы**, определяющие ядро вашей предметной области.

> Это уровень, где решаются _что_ и _зачем_, а не _как_.  
> Здесь нет зависимостей на фреймворки, базы данных или transport-слои.

---

## 🧩 Основные компоненты

| Компонент           | Описание                                                                    |
|---------------------|-----------------------------------------------------------------------------|
| **Entities**        | Основные бизнес-сущности и их методы (value-объекты, правила и инварианты). |
| **Errors**          | Доменные ошибки (например, `ErrInsufficientBalance`, `ErrNotFound`).        |
| **Enums/Constants** | Доменные типы и статусы (`UserStatus`, `OrderState`).                       |

---

## 🧱 Принципы

- **Отсутствие зависимостей.** `domain` ничего не знает о БД, HTTP, логгерах и т.п.
- **Чистая бизнес-логика.** Только то, что описывает поведение системы с точки зрения предметной области.
- **Контракты, а не реализации.** Только интерфейсы репозиториев и внешних сервисов.
- **Легко тестируется.** Всё можно покрыть unit-тестами без моков инфраструктуры.

---

## 📂 Пример структуры

domain/
├── user/
│ ├── entity.go // User, методы поведения
│ ├── repository.go // Интерфейс UserRepository
│ ├── errors.go // Доменные ошибки
│ └── value_objects.go // Email, PasswordHash, Role и т.п.
├── order/
│ └── ...
└── shared/
├── time_range.go // Общие value-объекты
└── status.go // Общие enum'ы и типы

## ✅ Пример: `User` `user/entity.go`

```go
package user

type User struct {
	ID        uuid.UUID
	Email     Email
	Password  PasswordHash
	Status    Status
	CreatedAt time.Time
}

func (u *User) Activate() error {
	if u.Status == StatusActive {
		return ErrAlreadyActive
	}
	u.Status = StatusActive

	return nil
}
```

## ✅ Пример: `Email` `user/value_objects.go`

```go
package user

type Email struct {
	value string
}

func NewEmail(v string) (Email, error) {
	if !isValidEmail(v) {
		return Email{}, ErrInvalidEmail
	}
	return Email{value: v}, nil
}

func (e Email) String() string {
	return e.value
}
```

## ✅ Пример: `TimeRange` `shared/time_range.go`

```go
package shared

import "time"

type TimeRange struct {
	From time.Time
	To   time.Time
}

func (tr TimeRange) Duration() time.Duration {
	return tr.To.Sub(tr.From)
}
```
