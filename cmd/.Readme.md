# 🚀 `cmd/`

## Назначение

Директория `cmd/` содержит **точки входа для различных бинарей** приложения.  
Каждая подпапка представляет отдельный исполняемый файл.

> В большинстве случаев это основной микросервис: `cmd/app/main.go`.

---

## 🧠 Структура

cmd/  
└── app/  
    └── main.go # HTTP/GRPC-сервис

---

## 🧼 Стиль: минимальный main.go

```go
package main

import "yourproject/internal/app"

func main() {
    app.Run()
}
```
