# 📁 `api/` — Контракты внешнего интерфейса микросервиса

Директория `api/` содержит **gRPC/Protobuf контракты**, которые определяют публичный интерфейс взаимодействия с микросервисом. Эти контракты являются единственным источником истины для других сервисов при обращении к текущему.

---

## 📦 Структура

api/  
├── service.proto # Основной gRPC-сервис: методы, сигнатуры  
└── types/  
└── types.proto # Общие типы, переиспользуемые в методах


---

## 🧠 Назначение

| Элемент             | Описание                                                                 |
|---------------------|--------------------------------------------------------------------------|
| `template.proto`    | Определяет RPC-методы, предоставляемые сервисом                         |
| `types/types.proto` | Описывает DTO/VO структуры, которые используются в сообщениях           |
| `option go_package` | Задаёт Go-модуль, в который будет сгенерирован код                      |
| `package`           | Пространство имён Protobuf, используется при импортах                   |
